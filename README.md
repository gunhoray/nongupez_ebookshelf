# 농업e지 eBook 뷰어

## 📖 프로젝트 개요

농업e지 교재들을 웹에서 쉽게 열람할 수 있는 접근성 친화적 eBook 뷰어입니다. PDF와 EPUB 형식을 지원하며, 특히 노년층을 고려한 사용자 친화적 인터페이스를 제공합니다.

## 🎯 주요 목표

- 웹 기반 PDF/EPUB 뷰어 구현
- 노년층 친화적인 UI/UX 디자인
- 접근성 향상 (글자 크기 조절, 색상 대비 등)
- 모바일 최적화 및 반응형 디자인
- 커스텀 목차 및 썸네일 기능

## 🛠️ 기술 스택

- **Frontend**: Next.js 15+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + Custom CSS
- **PDF Viewer**: PDF.js (pdfjs-dist)
- **EPUB Viewer**: react-reader (epub.js)
- **Package Manager**: npm
- **Deployment**: GitHub Pages / Vercel

## 🎨 UI/UX 디자인 특징

### 메인 페이지
- **그라데이션 배경**: 따뜻한 오렌지 계열 색상
- **책 카드 레이아웃**: 직관적인 그리드 형태
- **썸네일 이미지**: PDF 첫 페이지 썸네일 표시
- **반응형 디자인**: 모바일/태블릿 최적화
- **호버 효과**: 부드러운 애니메이션과 그림자
- **동적 제목**: 파일명 기반 자동 제목 생성

### 뷰어 페이지
- **커스텀 목차**: PDF별 맞춤 목차 (페이지 번호 제거)
- **사이드바 토글**: 모바일에서 숨김/표시 기능
- **줌 컨트롤**: 75% ~ 200% 줌 레벨 조절
- **페이지 네비게이션**: 이전/다음 버튼
- **PDF 다운로드**: 기능적인 다운로드 버튼
- **반응형 레이아웃**: 다양한 화면 크기 지원

## ✨ 주요 기능

### 1. 메인 페이지
- **동적 책 목록**: 파일명 기반 자동 제목 생성
- **썸네일 표시**: PDF 첫 페이지 썸네일 이미지
- **반응형 디자인**: 모바일/태블릿/데스크톱 최적화
- **부제목 시스템**: "저자" → "부제목" 변경

### 2. PDF 뷰어
- **커스텀 목차**: PDF별 맞춤 목차 (페이지 번호 제거)
- **모바일 최적화**: 사이드바 토글 기능
- **줌 컨트롤**: 75% ~ 200% 줌 레벨 조절
- **PDF 다운로드**: 기능적인 다운로드 버튼
- **키보드 네비게이션**: 화살표 키로 페이지 이동
- **안정성**: 렌더링 오류 처리 및 최적화

### 3. 모바일 최적화
- **반응형 레이아웃**: 다양한 화면 크기 지원
- **터치 친화적**: 모바일 터치 최적화
- **사이드바 토글**: 모바일에서 목차 숨김/표시
- **버튼 크기**: 터치하기 쉬운 버튼 크기

### 4. 접근성 개선
- **폰트**: Noto Sans KR 사용
- **색상**: 높은 대비의 오렌지 계열 (#ff8c42)
- **줌**: 75% ~ 200% 지원
- **키보드**: 화살표 키로 페이지 이동
- **안정성**: 페이지 전환 시 깜박거림 없는 부드러운 동작
